## Arrington Annotation Tool (AAT) v0.2.0

**Initial modular release**

This is the first public release of the clean, modular Arrington Annotation Tool (AAT).

### Highlights

- **Engine-powered dataset generation**: Use any pretrained YOLO model (`.pt`, `.engine`, `.onnx`) to automatically label images and produce ready-to-train YOLO detection datasets with confidence metadata.
- **AI-Assisted Viewer**: Full-featured desktop annotation tool with built-in "Suggest" functionality — load any model and get instant detection suggestions directly in the canvas for rapid review and correction.
- **Fully modular design**: Clean Python package (`import aat`) with pluggable inference, strong I/O primitives, and testable core.
- **CLI + Library**: `aat generate`, `aat view`, and a rich Python API.

### Key Components

- `aat.generate` — High-level dataset bootstrapping
- `aat.inference` — Pluggable detection engines (Ultralytics-backed by default)
- `aat.viewer` — Tkinter-based annotation UI with live AI assist
- `aat.io` — Robust YOLO label and shape handling (detect + polygon foundation)

### Installation

```bash
pip install "arrington-annotation-tool[ultralytics]"
```
